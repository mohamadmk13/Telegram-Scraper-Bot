# Telegram Product Scraper Bot

This project is a Telegram bot that allows users to search for products on **Newegg** and export detailed product information into an **Excel** file.

---

## Features

- Receive product name from the user via Telegram  
- Collect product details including:
  - Brand
  - Color
  - CPU
  - Memory
  - SSD
  - GPU
  - Price  
- Generate Excel files with all extracted information  
- Send Excel files directly to the user in Telegram  
- User-friendly interface with inline buttons and custom keyboards  
- Built using Python and free libraries  

---

## Installation & Run

make new config.py in the you folder code and copy the code from config example and put it in your config.py and change the API_Token with your real token
and run the code and test the bot
